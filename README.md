# cuesearch

**DISCLAIMER: THIS PROJECT IS STILL A WORK IN PROGRESS**

A web client for querying audio-post production data.

## Developing

### Install Dependencies

```bash
npm i
```

### Serve Client

```bash
npm run dev
```

### Run Mock Backend

```bash
chmod +x ./pingpong.py
./pingpong.py
```

## Building

```bash
npm run build
```
